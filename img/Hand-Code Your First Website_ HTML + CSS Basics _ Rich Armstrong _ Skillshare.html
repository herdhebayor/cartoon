    Project Description

    ASSIGNMENT: 

    You'll be learning how to write HTML and CSS by making a one-page website for your favourite cartoon hero or movie character, which you'll learn how to publish online for the world to see!

    DELIVERABLES: 

    Share a link to your character's hand-coded website! You're welcome to add your planning, and mention what you enjoyed, and what you found difficult :)

    A finished website includes the following:

    1. Your character’s profile image.
    2. The name of your character in a heading.
    3. An about paragraph.
    4. A numbered list of his/her characteristics.
    5. An unordered list of his/her friends and where to find them.
    6. A footer element containing a small amount of information.
